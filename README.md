# TripFlow AI

> AI-powered travel planning platform that creates personalized and optimized travel itineraries.

## 🌍 Vision

Help travelers spend less time planning and more time enjoying their journey.

## ✨ MVP Features

- Multi-day itinerary planning
- Multiple accommodations
- Route optimization
- Personalized travel style
- AI itinerary generation

## 🛠 Tech Stack

- Frontend: Next.js
- Backend: FastAPI
- Database: PostgreSQL
- Maps: Google Maps API
